# Cover Images

This directory stores cover art images for Yoto MYO cards.

## Image Requirements

According to Yoto's specifications (https://yoto.dev/myo/uploading-cover-images/):

- **Recommended sizes**: 661×1054 pixels or 784×1248 pixels
- **Aspect ratio**: 3:5 (vertical/portrait orientation)
- **Format**: PNG or JPEG
- **Purpose**: Custom cover images displayed on Yoto cards in the app and on players

## Files

- `default.png` - Default cover art (661×1054, permanent/non-deletable)
- `default_large.png` - Alternative large default (784×1248, permanent/non-deletable)

## Usage

Upload custom cover art through the web UI or API. The default cover art cannot be deleted.

Images can be:
1. Uploaded directly from your device
2. Fetched from a remote URL (using Yoto's autoconvert feature)
